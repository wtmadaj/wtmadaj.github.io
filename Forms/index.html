FORMS

This project explores creating and styling forms in HTML, CSS, and JavaScript. Use the navigation bar to check them out!


Basic

A basic form with simple styling. Pattern matching enforced.


SideNav

An enhanced version of the basic form.
Includes a styled drop-down menu and the navigation on the side


Multi-Step

A form that breaks each part into steps.
The user has to complete each step before moving on.
Includes dots to show the progress of the form.


Collapse

This form breaks the sections down into collapsible sections.
The user can open or close each section.
Also features a scroll to top button that appears when the user scrolls down.
